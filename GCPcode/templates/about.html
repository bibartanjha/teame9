Description

This site provides interesting and useful information on the NBA and the WNBA. Users can interact with this site to search up their favorite NBA players and teams. Users can also be kept up to date on the latest basketball scores and news.

The purpose of this site is to serve as a tool that NBA and WNBA fans can use to stay up-to-date regarding their favorite players and teams as well as the latest basketball stats and news.

The intended users are NBA and WNBA fans, or anyone who wants to learn more about these leagues.

Data

  • ESPN API
  • API-NBA
  • Sportradar: WNBA API

We are obtaining the data from existing NBA/WNBA and news APIs and using MongoDB as our database. The data is being requested through our main.py. The user may search up a certain player or team and our code requests the information about that searched element from these API’s and then displays the information to the user. For the user’s favorite players and teams information, they will be stored in MongoDB so that it is saved on their Favorite Player and Favorite Team pages for their specific account that they log in with on our website.

Tools

API-NBA: We are using the API-NBA to grab information about NBA players and teams.

WNBA-API: We are using the WNBA API provided by sportradar to grab information about WNBA players and teams.

ESPN API: We are using the ESPN API to grab news about the NBA and WNBA, as well as other statistics and season/trade predictions.

MongoDB: We are using MongoDB to store all the information we have gathered from our APIs, so we can easily access them via our program and the web.

Google Cloud Platform: We are using GCP to host our website, so anyone can have access to it and be able to use the website.

Flask: We are using Flask to have our python code work alongside the code for our static website.

Bootstrap: We are using Bootstrap CSS to add more stylization to our website, and make it visually appealing to the user.
